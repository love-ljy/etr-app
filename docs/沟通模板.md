# Agent 沟通模板库

## 快速复制使用

---

## 📢 进展同步模板

```markdown
【进展同步】@负责人

## 已完成
- [ ] 任务 1
- [ ] 任务 2

## 进行中
- [ ] 任务 3（预计完成时间）

## 遇到阻塞
- [ ] 问题描述
- [ ] 需要谁帮助

## 下一步
- [ ] 计划任务

时间：YYYY-MM-DD HH:MM
```

## 使用指南

### 何时使用哪个模板

| 场景 | 模板 |
|------|------|
| 定期汇报工作 | 进展同步 / 日报 |
| 需要别人配合 | 任务协调 |
| 完成工作移交 | 任务交接 |
| 遇到问题卡住 | 求助 |
| 需要拍板决定 | 决策请求 |
| 代码写完审核 | Code Review |
| 突发严重问题 | 紧急问题通知 |
| 分配任务给别人 | 任务分配 |
| 讨论技术细节 | 技术方案讨论 |
| 开会记录要点 | 会议记录 |

### 使用技巧

1. **复制模板** - 直接复制对应模板
2. **填充内容** - 替换方括号内容
3. **@相关人员** - 确保通知到人
4. **加上时间** - 重要消息标注时间
5. **跟进状态** - 发送后跟进响应

---

**好的沟通 = 清晰的结构 + 明确的需求 + 及时的响应**
