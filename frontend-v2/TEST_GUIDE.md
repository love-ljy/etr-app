## 🚀 快速开始

### 1. 启动开发服务器

```bash
cd frontend-v2
npm run dev
```

访问: http://localhost:3000

### 2. 准备测试钱包

#### 获取测试BNB
1. 访问 BSC测试网水龙头: https://testnet.bnbchain.org/faucet-smart
2. 输入你的钱包地址
3. 领取 0.5 BNB (用于gas费)

#### 获取测试USDT
**方式1: 使用水龙头函数**
- 合约地址: `0xEB4a0196124797a7580d2A260EBFF0bd845dc956`
- 调用 `faucet()` 函数
- 每次免费领取 10,000 USDT
- 冷却时间: 1小时

**方式2: 在BSCScan直接调用**
1. 访问: https://testnet.bscscan.com/address/0xEB4a0196124797a7580d2A260EBFF0bd845dc956#writeContract
2. 连接钱包
3. 点击 `faucet` 函数
4. 点击 `Write` 执行

### 3. 连接钱包

1. 点击页面右上角 "连接钱包" 按钮
2. 选择钱包 (MetaMask / Trust Wallet等)
3. 确认连接
4. 切换到 BSC测试网 (ChainID: 97)

## 🐛 问题排查

### 钱包连接问题

**问题**: 钱包连接失败
**解决方案**:
1. 确保MetaMask已安装
2. 确保在BSC测试网
3. 清除浏览器缓存
4. 刷新页面重试

**问题**: 网络错误
**解决方案**:
1. 切换到BSC测试网 (ChainID: 97)
2. 检查RPC配置
3. 尝试添加自定义RPC

### 交易问题

**问题**: 交易失败
**解决方案**:
1. 检查BNB余额 (gas费)
2. 检查代币余额
3. 检查代币批准状态
4. 增加Gas Limit

**问题**: 交易pending太久
**解决方案**:
1. 检查BSC测试网状态
2. 提高Gas Price
3. 取消并重新发送交易

### 合约交互问题

**问题**: 合约读取失败
**解决方案**:
1. 检查网络连接
2. 确认合约地址正确
3. 检查合约是否已部署
4. 查看浏览器控制台错误
