## 🔍 发现并修复的问题

### 问题1: 测试脚本质押代币错误
**问题描述**:
初始测试脚本误认为质押使用USDT，但实际StakingPool质押使用**ETR代币**

**修复方法**:
```javascript
// 错误：批准USDT
await usdt.approve(DEPLOYED_CONTRACTS.StakingPool, amount);

// 正确：批准ETR
await etr.approve(DEPLOYED_CONTRACTS.StakingPool, amount);
```

**修复结果**: ✅ 成功

---

### 问题2: 合约函数调用参数错误
**问题描述**:
- `calculatePendingReward(user, stakeId)` → 应该只传`stakeId`
- `compound.balances(user)` → 应该用`getCompoundBalance(user)`
- `referral.getUserStats(user)` → 应该用`getReferralStats(user)`

**修复方法**:
```javascript
// 修复收益计算
const pendingReward = await staking.calculatePendingReward(stakeId);

// 修复复利池余额查询
const balance = await compound.getCompoundBalance(user);

// 修复推荐统计查询
const stats = await referral.getReferralStats(user);
```

**修复结果**: ✅ 成功

---

### 问题3: 复利池统计返回值处理
**问题描述**:
`getPoolStats()`返回tuple `(totalPool, yieldRate, stakingPool)`，但测试脚本期望对象

**修复方法**:
```javascript
const stats = await compound.getPoolStats();
console.log("总存入:", ethers.formatEther(stats[0]));  // totalPool
console.log("日化收益率:", stats[1].toString());      // yieldRate
console.log("StakingPool:", stats[2]);                // stakingPool
```

**修复结果**: ✅ 成功
